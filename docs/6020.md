## 四、接口与线序

### 1. 电源接口（XT30）
- **额定电压**：24 V
- **线长**：500 mm

### 2. CAN信号接口
- **线序**（从上至下）：
  - A：红色 → CAN_H
  - B：黑色 → CAN_L
- **波特率**：1 Mbps
- **线长**：500 mm

### 3. PWM信号接口
- **线序**：
  - A：白色 → PWM/RX（接收）
  - B：灰色 → TX（发送）
  - C：黑色 → GND
- **支持信号**：
  - 频率：50 Hz
  - 脉宽范围：1000–2000 μs
- **线长**：500 mm

## 五、通信与控制模式

### 1. 控制方式支持
- **CAN 通信**（主控模式）  
- **PWM 通信**（模拟舵机模式）  
- **自动识别模式切换**

### 2. CAN 控制协议要点

- **控制报文 ID（电压/电流）**：
  - ID 1–4：`0x1FF`
  - ID 5–7：`0x2FF`
- **反馈报文 ID**：`0x204 + 电机ID`
- **反馈频率**：1 kHz
- **电压指令范围**：-25000 ~ +25000（单位无明确说明，代表内部电压设定值）
- **电流指令范围**：-16384 ~ +16384 → 对应 -3 A ~ +3 A

### 3. PWM 控制模式

| 模式 | 描述 |
|------|------|
| **位置模式**（默认） | 脉宽映射角度，可设中点（默认1500 μs）和转动范围 |
| **速度模式** | 脉宽映射转速（±100 rpm）：<br>1080 μs → -100 rpm，1520 μs → 0 rpm，1920 μs → +100 rpm |

> **PWM校准流程**：
> 1. 上电时输入高脉宽（1900–2100 μs）→ 绿灯常亮；
> 2. 3秒内切换至低脉宽（900–1100 μs）；
> 3. 成功：绿灯慢闪；失败：橙灯快闪。

## 六、ID 与拨码开关设置

### 1. 拨码开关定义
- **Bit 0–2（开关1–3）**：设置电机 ID（仅 1–7 有效，000 无效）
- **Bit 3（开关4）**：ON = 接入 CAN 终端电阻

### 2. ID 对应关系

| Bit[2:0] | 电机 ID | 反馈报文 ID | 控制报文 ID |
|----------|--------|--------------|--------------|
| 001 | 1 | 0x205 | 0x1FF |
| 010 | 2 | 0x206 | 0x1FF |
| 011 | 3 | 0x207 | 0x1FF |
| 100 | 4 | 0x208 | 0x1FF |
| 101 | 5 | 0x209 | 0x2FF |
| 110 | 6 | 0x20A | 0x2FF |
| 111 | 7 | 0x20B | 0x2FF |

## 七、状态指示灯说明

| 状态 | 指示灯行为 |
|------|-----------|
| **正常工作** | 绿灯每秒闪 N 次 → ID = N |
| **PWM通信正常** | 绿灯慢闪 |
| **PWM校准中** | 绿灯常亮 |
| **高温警告**（>100℃） | 橙灯每秒闪1次 |
| **CAN ID冲突** | 橙灯每秒闪2次 |
| **PWM信号无法识别** | 橙灯每秒闪3次 |
| **温度传感器异常** | 橙灯每秒闪4次 |
| **PWM校准失败** | 橙灯快闪 |
| **电流模式接收电压指令** | 橙灯常亮 |
| **供电过压** | 红灯每秒闪1次 |
| **温度过高**（>125℃） | 红灯每秒闪4次 |

> **注意**：异常状态下驱动器关闭输出。

## 九、安全与使用注意事项

- 严禁私自拆卸电机。
- 避免杂物进入转子内部。
- 安装务必稳固，使用匹配螺丝。
- 高扭矩运行时注意高温烫伤。
- 禁止触摸旋转部件，防止割伤。
- 接线前务必确认极性与线序正确。
